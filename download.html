Install

The dian-qc-uploader is designed as a self-contained application that can be use from anywhere. When invoking the java -jar command, simple pass the location of the dian-qc-uploader.jar file. If this file is in your currenty working directory, you can run:

java -jar dian-qc-uploader.jar <options>
