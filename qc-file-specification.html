The DIAN Quality Control Uploader requires that certain fields exist in the specific files.

Inclusion File

Details whether the participant meets the study criteria and the overall passing and payable status. File contains a single line for each session.

Required Columns

CSV Column
patientid
timestamp
initials
surgery
hemmorhage
dev_anomaly
lesion
atrophy
infarction
trauma
metallic
nph
edema
other
pass
pay_site
comments
